# SutBrute - R3X Edition

**"Sut silet, tembus celah."**  
Toolkit reconnaissance + brute force login dengan API key, proxy, dan multi-threading.

## Fitur
- **Recon**: SQLi, XSS, LFI, backup file exposure
- **Brute Force**: Multi-thread, CSRF auto-handling, proxy rotation
- **API Key**: Bawa `X-API-Key` tiap request (default: `SUTBRUTE-FREE-2026`)
- **Konfigurasi**: Ubah di `config/config.json`
